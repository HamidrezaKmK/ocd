# ocd
Optimized Cash Dashboard: A vibe coded financial manager


## Environment

This project uses [uv](https://docs.astral.sh/uv/). Install it, then:

```
uv sync
```

This creates a `.venv` with Python 3.11 and the project dependencies. Run commands with `uv run <cmd>` (e.g. `uv run python ...`), or activate the venv with `source .venv/bin/activate`.
